Authorization:Basic Sm9lbC5yaXZlcmE6Sm9lbFJfMjAxNw==

Obtener registros

debe enviar los siguientes campos para filtrar con el metodo "GET", todos son opcionales

limit, id, nombre, puesto

ejemplo: 

ver?limit=10

Insertar registro

Debe enviar una peticion tipo "POST" enviando un json con uno o mas registros a insertar 

ejemplo:
            agregar
                [{
                    "nombre": "lupita",
                    "fecha_nacimiento": "10-02-1990",
                    "puesto": "administradora"
                },{
                    "nombre": "ana",
                    "fecha_nacimiento": "10-02-1990",
                    "puesto": "mantenimiento"
                }]

Modificar registro

Debe enviar una peticion tipo "PUT" enviando un json con los datos y id de la persona a modificar

        ejemplo: 
        modificar
                [{
                    "id"= 5,
                    "nombre": "lupita",
                    "fecha_nacimiento": "10-02-1990",
                    "puesto": "mantenimiento"
                }]
        nota: el "id" los demas campos son opcionales

Eliminar registro

Debe enviar una peticion tipo "DELETE" enviando id de la persona a modificar

            ejemplo:
            eliminar?id=1
